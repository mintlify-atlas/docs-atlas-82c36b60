wgpu is a cross-platform, safe, pure-Rust graphics API. It runs natively on Vulkan, Metal, D3D12, and OpenGL, and on top of WebGL2 and WebGPU on WebAssembly. The API is based on the [WebGPU standard](https://gpuweb.github.io/gpuweb/) and serves as the core of the WebGPU integration in Firefox, Servo, and Deno.

## Why wgpu?

<CardGroup cols={2}>
  <Card title="Cross-platform by design" icon="globe">
    One API targets Vulkan, Metal, D3D12, OpenGL, WebGL2, and WebGPU. Write once, run anywhere.
  </Card>
  <Card title="Safe Rust API" icon="shield-halved">
    No unsafe code required at the user level. wgpu-core handles all validation and resource tracking.
  </Card>
  <Card title="WebGPU standard" icon="file-contract">
    Based on the W3C WebGPU specification. Knowledge transfers between native and web targets.
  </Card>
  <Card title="WGSL shader language" icon="microchip">
    First-class support for WGSL, plus SPIR-V and GLSL via Naga, the built-in shader translator.
  </Card>
</CardGroup>

## Get started in three steps

<Steps>
  <Step title="Add wgpu to your project">
    Add `wgpu` to your `Cargo.toml`:

    ```toml Cargo.toml
    [dependencies]
    wgpu = "29"
    pollster = "0.4"
    ```
  </Step>
  <Step title="Create an Instance and request a Device">
    The `Instance` is your entry point. From it you enumerate adapters and open a logical `Device`:

    ```rust main.rs
    let instance = wgpu::Instance::default();
    let adapter = pollster::block_on(instance.request_adapter(
        &wgpu::RequestAdapterOptions::default(),
    )).unwrap();
    let (device, queue) = pollster::block_on(adapter.request_device(
        &wgpu::DeviceDescriptor::default(),
    )).unwrap();
    ```
  </Step>
  <Step title="Submit GPU work">
    Encode commands and submit them to the `Queue`:

    ```rust main.rs
    let mut encoder = device.create_command_encoder(
        &wgpu::CommandEncoderDescriptor { label: Some("encoder") }
    );
    // record render or compute passes here
    queue.submit(std::iter::once(encoder.finish()));
    ```
  </Step>
</Steps>
